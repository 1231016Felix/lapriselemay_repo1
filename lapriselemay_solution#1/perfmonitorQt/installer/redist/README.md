# Visual C++ Redistributable

This folder should contain the Visual C++ Redistributable installer for the application installer.

## Download

Download the latest Visual C++ Redistributable (x64) from Microsoft:
https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist

Direct link (may change):
https://aka.ms/vs/17/release/vc_redist.x64.exe

## File Required

Place the downloaded file here as:
- `VC_redist.x64.exe`

This file is required to build the installer with Inno Setup.
